KEY DESIGN DECISIONS
================================================================================

1. Dysregulation Criterion (biomarker_validation.py, line 125)
   AND logic: cv > 0.5 AND max_fold_change > 1.5
   - Both criteria must be satisfied for a gene to be classified as dysregulated
   - CV (coefficient of variation) measures expression variability
   - Fold-change measures magnitude of expression shift

2. Test Data Configuration (test_phase3_integration.py, line 51)
   - Baseline noise: SD=3.0 (provides CV ~0.4-0.5)
   - Disease module dysregulation: +4±1.0 (provides strong signal)
   - This configuration ensures sufficient dysregulation to meet validation thresholds
   - Critical for test_cross_scale_consistency to detect biomarkers

3. SIS Dynamics Parameters
   - n_steps=500 (sufficient for convergence)
   - n_runs=50-100 (robust stochastic sampling)
   - Persistence threshold: 75th percentile or min_persistence=0.3
   - Biomarkers ranked by persistence score ∈ [0, 1]

4. Multi-Module Support
   - Pipeline supports multiple disease modules simultaneously
   - Each module has independent parameter extraction and dynamics
   - Results aggregatable across modules for systems-level analysis
