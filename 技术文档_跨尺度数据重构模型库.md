# 跨尺度数据重构模型库 技术文档

## 1. 系统概述

### 1.1 项目定位

本系统是一个基于 Web 的跨尺度生物网络分析平台，实现从分子层（基因互作）到通路层（信号通路活性）、组织层（共表达网络）再到群体层（传播动力学）的多尺度网络建模与可视化。

系统采用 Python + Gradio 框架开发，后端集成 NetworkX 图算法库和 Plotly 可视化引擎，前端提供交互式操作界面。

### 1.2 技术栈

| 层级 | 技术 | 版本要求 |
|------|------|---------|
| Web 框架 | Gradio | ≥ 4.0.0 |
| 可视化 | Plotly | ≥ 5.18.0 |
| 图算法 | NetworkX | ≥ 3.0 |
| 数据处理 | Pandas / NumPy | ≥ 2.0 / ≥ 1.24 |
| 统计分析 | SciPy / statsmodels | ≥ 1.10 / ≥ 0.14 |
| 机器学习 | scikit-learn | ≥ 1.3.0 |
| 并行计算 | joblib | ≥ 1.3.0 |

### 1.3 数据来源

| 数据集 | 规模 | 说明 |
|--------|------|------|
| TCGA-COAD RNA-seq | 14,521 基因 × 257 样本 | 结直肠腺癌基因表达谱 |
| TCGA-COAD miRNA-seq | 619 miRNA × 257 样本 | miRNA 表达谱 |
| TCGA-COAD 临床数据 | 257 样本 × 5 变量 | 年龄、性别、分期等 |

## 3. 核心算法

### 3.1 通路影响力评分（CV 加权法）

**模块**: `disease_tracing.py` → `DiseaseTracer.get_pathway_ranking()`

对给定疾病 d，计算每条通路 p 的影响力：

$$
\text{PathwayScore}(p, d) = |G_p \cap G_d| \times \overline{CV}
$$

其中：

$$
CV(g) = \frac{\sigma(\text{expr}_g)}{|\mu(\text{expr}_g)| + \varepsilon}, \quad \varepsilon = 10^{-6}
$$

- $G_p$：通路 p 包含的基因集合
- $G_d$：疾病 d 关联的基因集合
- $CV(g)$：基因 g 在 TCGA-COAD 257 个样本中的表达变异系数
- $\overline{CV}$：$G_p \cap G_d$ 中所有基因的 CV 均值

**物理意义**：通路中与疾病重叠的基因越多，且这些基因跨样本的表达变异越大，则该通路对疾病的影响力越高。

### 3.2 节点属性熵（NAE）

**模块**: `disease_tracing.py` → `DiseaseTracer.get_gene_module()`

对基因 g 构建 4 维属性向量，计算信息熵：

$$
NAE(g) = -\sum_{k=1}^{4} p_k \cdot \log_2(p_k)
$$

属性向量：

| 维度 | 属性 | 计算方式 |
|------|------|---------|
| $a_1$ | 度中心性 | $DC(g) = \deg(g) / (N-1)$ |
| $a_2$ | 介数中心性 | 经过 g 的最短路径比例 |
| $a_3$ | 表达方差比 | $\text{Var}(g) / \max(\text{Var})$ |
| $a_4$ | 通路参与度 | g 参与的通路数 / 总通路数 |

归一化：$p_k = a_k / \sum_k a_k$

**物理意义**：NAE 越高，说明基因在多个维度上同时具有较高的重要性，是通路功能的核心调控因子。

### 3.3 miRNA-基因负相关调控筛选

**模块**: `disease_tracing.py` → `DiseaseTracer.get_gene_expression_profile()`

对目标基因 j，从 619 条 miRNA 中筛选：

$$
\text{target if } r(miRNA_i, Gene_j) < -0.3 \text{ 且 } p < 0.05
$$

- $r$：Pearson 相关系数，基于 257 个样本的表达值计算
- 原理：miRNA 通过结合 mRNA 3'UTR 抑制翻译，因此 miRNA 与靶基因表达呈负相关

### 3.4 GSVA 通路活性评分

**模块**: `pathway_activity.py` → `PathwayActivityScorer.score_gsva()`

对每个通路 p、每个样本 s：

1. 提取通路基因的表达矩阵 $E_p$（基因 × 样本）
2. 按基因做 Z-score 标准化：$\hat{E}_{g,s} = (E_{g,s} - \mu_g) / \sigma_g$
3. 通路活性：$\text{Score}(p, s) = \text{mean}(\hat{E}_{:,s})$

输出：通路活性矩阵（347 通路 × 257 样本），值域约 [-3, 3]。

### 3.5 差异通路分析

**模块**: `differential_pathway_analysis.py` → `DifferentialPathwayAnalysis.compare_by_group()`

按临床变量（年龄组/性别/分期）分组，对每条通路做统计检验：

| 分组数 | 检验方法 | 统计量 |
|--------|---------|--------|
| 2 组 | Student's t-test | t 值 |
| ≥3 组 | One-way ANOVA | F 值 |

多重检验校正：Benjamini-Hochberg FDR，$\alpha = 0.05$。

### 3.6 枢纽基因综合评分

**模块**: `hub_gene_identification.py` → `HubGeneIdentifier.get_top_hub_genes()`

$$
\text{HubScore}(g) = 0.4 \times DC(g) + 0.3 \times BC(g) + 0.3 \times EV(g)
$$

- $DC$：度中心性（归一化到 [0,1]）
- $BC$：介数中心性（归一化到 [0,1]）
- $EV$：表达方差（归一化到 [0,1]）

### 3.7 Hill 函数跨尺度参数传递

**模块**: `cross_scale_engine.py` → `CrossScaleEngine._derive_population_params()`

将分子/细胞层的网络拓扑信号映射为群体层 SIS 模型参数：

$$
\beta(s) = \beta_{min} + (\beta_{max} - \beta_{min}) \times \frac{s^n}{s^n + K^n}
$$

| 参数 | 值 | 说明 |
|------|-----|------|
| $\beta_{min}$ | 0.01 | 最低感染率 |
| $\beta_{max}$ | 0.20 | 最高感染率 |
| $K$ | 0.3 | 半饱和常数 |
| $n$ | 2 | Hill 系数 |
| $s$ | $(d_{mol} + c_{cell})/2$ | 分子密度与细胞聚类系数的均值 |

### 3.8 WGCNA 共表达网络分析

**模块**: `wgcna_analysis.py` → `WGCNAAnalyzer`

**软阈值选择**：对候选幂 $\beta \in [1, 20]$，计算无标度拓扑拟合度 $R^2$，选择使 $R^2 > 0.9$ 的最小 $\beta$。

**拓扑重叠矩阵（TOM）**：

$$
TOM_{ij} = \frac{\sum_u w_{iu} \cdot w_{uj} + w_{ij}}{\min(k_i, k_j) + 1 - w_{ij}}
$$

其中 $w_{ij} = |cor_{ij}|^\beta$，$k_i = \sum_u w_{iu}$。

**模块检测**：基于 $1 - TOM$ 距离矩阵做层次聚类，动态剪切树获得共表达模块。

**模块特征基因**：模块内基因表达矩阵的第一主成分（PCA PC1），代表模块的整体表达模式。

### 3.9 SIS 传播动力学模型

**模块**: `social_network_sim.py` → `SocialNetworkSimulator`

**网络生成**（随机块模型）：

$$
p_{in} = \frac{Z_{in}}{N/c - 1}, \quad p_{out} = \frac{k_1 - Z_{in}}{N - N/c}
$$

- $N$：节点数，$c$：社区数，$k_1$：平均度，$Z_{in}$：社区内平均连接数

**SIS 状态转移**（每个时间步）：

- 感染节点恢复概率：$P(\text{recover}) = \gamma$
- 易感节点感染概率：$P(\text{infect}) = 1 - \prod_{j \in \mathcal{N}(i)} (1 - \beta \cdot I_j)$

其中 $I_j \in \{0, 1\}$ 为邻居 j 的感染状态。

### 3.10 Louvain 社区检测

**模块**: `disease_module_detection.py` → `CommunityDetector.detect_communities_louvain()`

最大化模块度：

$$
Q = \sum_c \left[ \frac{e_c}{m} - \left(\frac{a_c}{2m}\right)^2 \right]
$$

- $e_c$：社区 c 内的边数
- $a_c$：社区 c 内节点的度数之和
- $m$：总边数

## 4. 数据流

### 4.1 疾病多尺度溯源流程

```
用户选择疾病
    │
    ├──→ get_pathway_ranking()
    │       输入: gene_disease.tsv + TCGA 表达
    │       算法: CV 加权评分 (3.1)
    │       输出: 通路排名 DataFrame
    │
    ├──→ create_pathway_universe()
    │       输入: pathway.tsv (347 通路)
    │       构建: 共享基因 ≥50 的通路间网络
    │       输出: Plotly 网络图 (347 节点, ~643 边)
    │
    └──→ create_gene_universe()
            输入: TCGA 表达 (top 400 高方差基因 + 通路基因)
            构建: 共表达网络 (|r| > 0.5)
            输出: Plotly 网络图 (~490 节点)
                  三层着色: 灰=背景, 青=通路基因, 绿=疾病基因
```

### 4.2 跨尺度级联流程

```
选择疾病
    │
    ├──→ 分子层分析
    │       构建基因互作网络
    │       计算拓扑指标 (密度, 聚类系数, 平均度)
    │       识别 Hub 基因 (NAE 排序)
    │
    ├──→ 细胞/组织层分析
    │       Hub 基因作为种子节点
    │       MRNetB 网络推断 (互信息)
    │       计算细胞层聚类系数
    │
    └──→ 群体层参数映射
            s = (mol_density + cell_clustering) / 2
            β = Hill(s)
            输出: SIS 模型参数 {β, γ, N, c}
```

### 4.3 通路活性分析流程

```
点击"开始分析"
    │
    ├──→ PathwayActivityScorer.score_gsva()
    │       输入: 14,521 基因 × 257 样本 + 347 通路定义
    │       输出: 347 × 257 活性矩阵
    │
    ├──→ DifferentialPathwayAnalysis.compare_by_group()
    │       输入: 活性矩阵 + 临床分组 (Age_Group/Gender/Stage)
    │       输出: 差异通路表 (pathway, pvalue, padj, significant)
    │
    └──→ HubGeneIdentifier.get_top_hub_genes()
            输入: 通路基因 + 基因网络
            输出: 枢纽基因表 (gene, hub_score, degree, betweenness)
```
