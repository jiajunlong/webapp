🧬 Model 1: GENE INTERACTION NETWORK
   Scale: Molecular | Algorithm: Graph theory + degree centrality
   Input: Gene-disease database | Output: Network topology, Hub genes

🧬 Model 2: GENE REGULATORY NETWORK  
   Scale: Molecular | Algorithm: Directed graph + regulatory cascade
   Input: Gene-disease DB | Output: TF rankings, regulatory pathways

🧬 Model 3: IS COEFFICIENT
   Scale: Molecular | Algorithm: Influence score computation
   Input: Gene-pathway mapping | Output: Pathway impact rankings

🔬 Model 4: MRNetB (Network Inference)
   Scale: Cellular/Tissue | Algorithm: MI-based network reconstruction
   Input: TCGA expression data | Output: Gene association networks
   Formula: Weight(i,j) = MI(i,j) - max(min(MI(i,:), MI(j,:)))

🔬 Model 5: NETWORK FLOW ENTROPY (MFE)
   Scale: Cellular/Tissue | Algorithm: Information-theoretic entropy
   Input: Gene expression + network topology | Output: Node entropy values

👥 Model 6: SIS EPIDEMIC DYNAMICS
   Scale: Population | Algorithm: SIS transmission on networks
   Input: Network params + β/γ rates | Output: Infection curves
   Formula: P(S→I) = 1-(1-β)^k, P(I→S) = γ
