LAUNCHING THE APPLICATION
================================================================================

1. Ensure you have the required data files:
   - data/TCGA-COAD/filtered_hiseq_data.csv (gene expression)
   - data/TCGA-COAD/filtered_clinical.csv (clinical data)
   - data/TCGA-COAD/filtered_miRNA_with_names.csv (miRNA expression)
   - data/gene_disease.tsv (gene-disease associations)
   - data/pathway(基因名映射版).tsv (pathway-gene mappings)

2. If you don't have test data, generate it:
   $ python3 generate_tcga_test_data.py

3. Launch the application:
   $ python3 app_full.py
   
   Or with Gradio options:
   $ gradio app_full.py --server_name 0.0.0.0 --server_port 7860

4. Open browser to: http://localhost:7860

================================================================================
USER WORKFLOW
================================================================================

PATHWAY ANALYSIS (Phase 1)
  1. Navigate to: 🧬 通路活性分析
  2. Select clinical grouping variable
  3. Choose scoring method (GSVA or Mean)
  4. Click "运行通路分析"
  5. View pathway heatmap, differential analysis, hub genes

NETWORK MEDICINE (Phase 2)
  1. Navigate to: 🔗 网络医学分析
  2. Choose sub-analysis:
     a) Disease Module Detection:
        - Select disease
        - Set minimum module size
        - Run detection
     b) WGCNA Analysis:
        - Select clinical trait
        - Run co-expression network analysis
     c) miRNA Analysis:
        - Set correlation threshold
        - Analyze regulatory networks

BIOMARKER DISCOVERY (Phase 3)
  1. Navigate to: 🔬 SIS生物标志物发现
  2. Select disease module (requires Phase 2 results)
  3. Configure parameters:
     - β (transmission rate)
     - γ (recovery rate)  
     - Simulation steps and runs
  4. Run analysis
  5. View ranked biomarkers and validation results

================================================================================
TROUBLESHOOTING
================================================================================

Q: "Data not loaded" error
A: Verify data files exist in data/ directory
   Run: python3 generate_tcga_test_data.py to create test data

Q: Phase 2 analysis runs but returns empty results
A: Ensure:
   - Gene expression data loaded (filtered_hiseq_data.csv)
   - Gene-disease associations present (gene_disease.tsv)
   - Minimum parameters configured
   - Wait for progress bar to complete (0% → 95%)

Q: Phase 3 says "No modules found"
A: Phase 3 requires Phase 2 results first:
   1. Run Phase 2 disease module detection
   2. Ensure modules are saved
   3. Then try Phase 3 analysis

Q: Slow performance
A: Phase 2 & 3 use stochastic simulation (100 runs × 1000 steps)
   Expected: 30-60 seconds per analysis
   Reduce n_steps/n_runs if needed for testing
