1. PARAMETER EXTRACTION (parameter_extraction.py)
   ─────────────────────────────────────────────────
   Purpose: Bridge Phase 2 disease modules to Phase 3 SIS dynamics
   
   Key Components:
   • extract_transmission_rate(module_name, connectivity_weight=0.6, variance_weight=0.4)
     - Computes β from network topology and expression variance
     - Formula: β = min(1, α * connectivity * expression_variability)
     - Range: [0, 1]
   
   • extract_recovery_rate(module_name, redundancy_factor=None)
     - Computes γ from pathway redundancy
     - Formula: γ = 1 / (1 + redundancy_factor)
     - Higher redundancy → lower γ (disease persists)
     - Range: [0.01, 1]
   
   • extract_initial_infection(module_name, disease_group=None, fc_threshold=0.5, pval_threshold=0.05)
     - Determines I₀ (initial dysregulation) from differential expression
     - Uses t-test between disease and control groups
     - I₀[g] ∝ |log2 fold-change| for significant genes
     - Range: [0, 1]^n
   
   • extract_network_structure(module_name, correlation_threshold=0.3)
     - Returns adjacency matrix (pre-computed or from co-expression)
     - Supports weighted networks
   
   • extract_all_parameters(module_name, disease_group=None)
     - Extracts all SIS parameters for a single module
     - Returns dict: {beta, gamma, initial_infection, adjacency, n_genes, n_edges}

   Constructor:
   ParameterExtractor(disease_modules, expression_data, clinical_data=None, adjacency_matrices=None)


2. SIS NETWORK PROPAGATION (sis_network_propagation.py)
   ───────────────────────────────────────────────────────
   Purpose: Run stochastic SIS dynamics to identify persistent biomarkers
   
   Key Concept: 
   • Nodes = Genes in disease module
   • "Infection" = Dysregulation (abnormal expression)
   • Transmission = Dysregulation spreads through network with probability β×A[i,j]
   • Recovery = Infected genes recover with probability γ
   • Biomarkers = Genes with high persistence scores

   Algorithm:
   For each run:
     I = I₀ (initialize infection)
     For each timestep:
       - Transmission: infected node i infects neighbors j with prob β×A[i,j]×I[i]
       - Recovery: infected nodes recover with prob γ
       - Record infection state

   Key Methods:
   • run_dynamics(n_steps=1000, n_runs=100, random_seed=None)
     - Returns dict with infection_history, persistence_scores, biomarkers, statistics
     - infection_history: (n_runs × n_steps × n_nodes)
     - persistence_scores: fraction of timesteps each gene remains infected
   
   • get_biomarker_table(top_n=20)
     - Returns DataFrame with ranked biomarkers and network metrics
     - Columns: rank, gene, persistence, degree, strength, I0
   
   • compute_weighted_biomarker_score(centrality_weights=None)
     - Combines persistence + network centrality
     - Score = 0.6 × persistence + 0.4 × centrality

   Constructor:
   SISNetworkPropagation(adjacency_matrix, gene_names, parameters)
   where parameters = {'beta': float, 'gamma': float, 'initial_infection': np.ndarray}


3. BIOMARKER VALIDATION (biomarker_validation.py)
   ───────────────────────────────────────────────
   Purpose: Validate SIS-predicted biomarkers against biological and clinical data
   
   Validation Methods:
   • validate_expression_changes(disease_stage_col=None)
     - Checks if biomarkers are dysregulated in disease samples
     - Criteria: CV > 0.5 AND max_fold_change > 1.5
     - Returns DataFrame with dysregulation metrics
   
   • validate_clinical_correlation(outcome_variable)
     - Tests correlation between biomarker expression and clinical outcome
     - Computes Pearson and Spearman correlations
     - Tests significance (p < 0.05)
   
   • compute_biomarker_signature_score(genes=None, method='mean')
     - Creates multi-gene signature score
     - Methods: 'mean' (simple average), 'weighted' (by scores), 'pca' (first PC)
     - Returns normalized scores ∈ [0, 1]
   
   • compare_with_literature(known_biomarkers=None)
     - Compares predictions with known disease biomarkers
     - Returns overlap statistics and novel genes

   Constructor:
   BiomarkerValidator(predicted_biomarkers, biomarker_scores, expression_data, clinical_data)

KEY DESIGN DECISIONS
================================================================================

1. Test Data Generation
   - Baseline noise: SD = 3.0 (strong signal)
   - Disease dysregulation: +4 ± 1.0 (fold-change ~1.66-2.3x)
   - Result: Meets dysregulation criteria (CV > 0.5 AND FC > 1.5)
   - 80 healthy samples + 40 disease samples (typical case-control)

2. Dysregulation Detection Thresholds
   - Coefficient of Variation (CV) > 0.5: variability criterion
   - Fold Change (FC) > 1.5: magnitude criterion
   - BOTH must be satisfied (AND logic)
   - Rationale: Distinguishes true dysregulation from noise

3. Persistence Score Calculation
   - persistence = (# timesteps infected) / (n_runs × n_steps)
   - Biomarker threshold: 75th percentile or minimum 0.3
   - Higher persistence = more robust biomarker

4. Parameter Extraction Strategy
   - Per-module extraction allows multi-module diseases
   - Clinical grouping (disease_status='disease') enables targeted learning
   - Network structure can be pre-computed (PPI) or derived (co-expression)

1. Basic Workflow (Complete Pipeline)
   ───────────────────────────────────

   from parameter_extraction import ParameterExtractor
   from sis_network_propagation import SISNetworkPropagation
   from biomarker_validation import BiomarkerValidator

   # Step 1: Extract SIS parameters from disease module
   extractor = ParameterExtractor(
       disease_modules={'Heart_Disease': ['GENE1', 'GENE2', ...]},
       expression_data=expr_df,  # genes × samples
       clinical_data=clinical_df,
       adjacency_matrices={'Heart_Disease': ppi_matrix}
   )
   
   params = extractor.extract_all_parameters(
       module_name='Heart_Disease',
       disease_group='disease_status'
   )

   # Step 2: Run SIS dynamics
   propagation = SISNetworkPropagation(
       adjacency_matrix=params['adjacency'],
       gene_names=['GENE1', 'GENE2', ...],
       parameters={
           'beta': params['beta'],
           'gamma': params['gamma'],
           'initial_infection': params['initial_infection']
       }
   )
   
   propagation.run_dynamics(n_steps=500, n_runs=100)

   # Step 3: Get top biomarkers
   biomarker_df = propagation.get_biomarker_table(top_n=20)
   biomarkers = biomarker_df['gene'].tolist()

   # Step 4: Validate predictions
   validator = BiomarkerValidator(
       predicted_biomarkers=biomarkers,
       biomarker_scores=biomarker_df['persistence'].values,
       expression_data=expr_df,
       clinical_data=clinical_df
   )
   
   expr_validation = validator.validate_expression_changes('disease_status')
   clinical_validation = validator.validate_clinical_correlation('survival_days')
   
   signature_score = validator.compute_biomarker_signature_score()
