USAGE INSTRUCTIONS
================================================================================

PHASE 1: Pathway Activity Analysis
  1. Click Tab 4: 🧬 通路活性分析
  2. Select pathways from dropdown
  3. Choose disease/healthy groups
  4. Configure analysis parameters:
     - Scoring method (GSVA or mean)
     - P-value threshold
     - FDR correction
  5. Click "运行分析" (Run Analysis)
  6. View: Activity heatmap, differential plots, hub genes

PHASE 2: Network Medicine Analysis
  Sub-tab A - Disease Module Detection:
    1. Enter disease name
    2. Set minimum module size
    3. Set correlation threshold
    4. Click "运行分析"
    5. View: Module table, network metrics

  Sub-tab B - WGCNA Analysis:
    1. Select clinical trait
    2. Set minimum module size
    3. Set top N hub genes to display
    4. Click "运行分析"
    5. View: WGCNA modules, trait correlations, hub genes

  Sub-tab C - miRNA Analysis:
    1. Set correlation threshold
    2. Set p-value threshold
    3. Choose prediction method
    4. Click "运行分析"
    5. View: Hub miRNAs, pathway coverage, regulatory modules

PHASE 3: SIS Biomarker Discovery
  1. Click Tab 6.7: 🔬 SIS生物标志物发现
  2. Select disease module from Phase 2
  3. Configure SIS parameters:
     - Transmission rate (β)
     - Recovery rate (γ)
     - Initial infection (I₀)
  4. Set simulation parameters:
     - Number of runs
     - Number of timesteps
  5. Click "运行分析"
  6. View: Biomarker table, infection dynamics, validation results
